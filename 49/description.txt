题目描述

DG上幼儿园的时候，最爱做的一个游戏叫“分花生”，不仅是因为DG爱吃花生的缘故，更重要的是每次“分花生”的时候，他总能近距离观察班上的其他MM。所以在此我也郑重的向大家介绍下当年DG所做的“分花生”游戏。
游戏中有三种操作（每种操作可以执行无限次）：
a i: 老师分给第i个学生一枚花生
b i: 第i个学生将他所有的花生退给老师
c i j: 学生i和学生j交换手中的花生
游戏开始之前老师会从上述三种操作中选取k个作为一个组，称之游戏步骤。然后老师找齐班上的n个学生，带上一大包花生（无限多），并叫学生们重复m次游戏步骤。每次游戏之后，DG都迫切的希望知道游戏结束后每个学生手中花生的数目，知己知彼嘛，一来可以解解馋，二来可以用花生讨好别的MM。
输入

输出包含多组测试数据，以“0 0 0”结束
每组测试数据的第一行包含三个整数n,m,k (m≤1,000,000,000, n≤100, k≤100)
之后包含k行，每行一个操作
输出

输出为一行，包含n个整数，依次表示n个学生手中花生的数目。

样例输入

3 1 6
a 1
a 2
a 2
c 1 2
a 3
b 2
0 0 0
